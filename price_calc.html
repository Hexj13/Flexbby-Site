Внедрение и лицензирование
системы Flexbby One

Мы разработали несколько вариантов внедрения системы Flexbby One, чтобы вы выбрали лучший по оптимальному уровню расходов. Теперь легко и просто определить стоимость нашего решения для вас.

Приобретая технологию Flexbby One, Вы оплачиваете внедрение и лицензирование. И если цена инсталляции рассчитывается нами индивидуально с учетом ваших технических требований, то тариф за лицензирование Вы можете определить самостоятельно в нашем специальном калькуляторе лицензий.
Калькулятор определения стоимости лицензий
Для начала необходимо выбрать тип лицензии, который предполагает удобный для вас способ установки системы Flexbby One:

  • Локально, On-Premises (ваш сервер) / Лицензия On-Prem
  • В нашем облаке Flexbby Cloud (сервер компании Flexbby) / Лицензия Flexbby Cloud
  • В облаке Microsoft Azure (сервер компании Microsoft) / Лицензия For Azure
